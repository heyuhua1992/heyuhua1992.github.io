把vue全家桶+express+mongodb 部署到服务器

发表于 2018-09-13 | 更新于 2018-09-14 | 分类于 其他 | 评论数： | 阅读次数：
本文字数： 4.5k | 阅读时长 ≈ 4 分钟

软件版本

window 10 开发环境下：

  • Vue：2.9.6

  • Node：v8.11.3

  • Express：4.16.0

  • MongoDb：MongoDB shell version v4.0.2

在centos7 服务器环境进行部署：

  • Node：v8.12.0
  • pm2：3.1.2
  • nginx：nginx/1.14.0

前置条件

  1. 已经安装node
  2. 有自己的域名

服务器部署

  1. 把用express搭建的node服务器端打包成zip文件

  2. 把zip文件传到服务器的 /heyuhua/mall 目录下（需要预先建立好文件目录）

  3. 服务器端安装zip解压缩 ： yum install zip unzip

  4. 解压缩zip文件 unzip server.zip (tips: 打包成zip的时候把node_modules文件也打包进去了，不用再执行npm install了，否则需要执行npm install)

  5. vue项目运行npm run build ,把生成的文件打包成zip文件，传输到服务器的/heyuhua/default/mall 目录下（需要预先建立好文件目录），解压缩，得到index.html与static文件夹。(tips: 打包之前把vue项目中的config/index.js文件的assetsPublicPath设置为空：assetsPublicPath: ‘’.部署完后，访问的时候，路径会出现两个斜杆–> // )

  6. 服务器安装pm2 ：npm install pm2 -g

  7. 尝试以pm2启动服务器

    • 进入服务器的根目录下的bin文件
    • pm2 start www
    • 关闭：pm2 delete 0 (对应的id号)

  8. 安装nginx

    1. 添加源： sudo rpm -Uvh http://nginx.org/packages/centos/7/noarch/RPMS/nginx-release-centos-7-0.el7.ngx.noarch.rpm

    2. 更新源：sudo yum update

    3. 安装：sudo yum install -y nginx

    4. 启动：nginx

    5. 启动Nginx并设置开机自动运行（必须先关闭后运行以下语句，关闭：pkill -9 nginx）
